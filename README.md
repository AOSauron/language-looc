### Installation without using Atom

  - Close **Atom**
  - `cd ~/.atom/packages`
  - `git clone https://github.com/AOSauron/language-looc.git`
  - `npm install`
  - You can now restart Atom.

### Update the package

  - Close **Atom**
  - `cd ~/.atom/packages/language-looc`
  - `git pull origin master`
  - `npm install`
  - You can now restart Atom.
